En este espacio voy guardando avances, pruebas y versiones del proyecto que estoy construyendo, lo cual me permite ver mi progreso y corregir errores con el tiempo. También funciona como portafolio donde puedo mostrar lo que he aprendido durante mi estancia en la carrera de arte y comunicacion digitales, además de servir como respaldo de mi trabajo y una herramienta para colaborar con otros compañeros si es necesario.!_😃⭐
